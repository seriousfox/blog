I ran into a problem whilst trying to titleizeÂ a string containing an acronym. For example:

{% highlight ruby %} "building an API".titleize {% endhighlight %}

Will output

{% highlight ruby %} => "Building An Api" {% endhighlight %}

Not exactly what I was after but with a small change to the Inflector class you can define your own acronyms.

config/initializers/inflections.rb

{% highlight ruby %} ActiveSupport::Inflector.inflections do |inflect| inflect.acronym "API" end {% endhighlight %}

Now:

{% highlight ruby %} "Building An API".titleize {% endhighlight %}
